# fresh project

## Usage

Start the project:

```
deno task start
```

This will watch the project directory and restart as necessary.

------
## Note
### Create a project
```
deno run --allow-read --allow-write=./md-sns2 --allow-net=fresh.deno.dev --allow-run=$HOME/.deno/bin/deno -r https://fresh.deno.dev md-sns2
```
